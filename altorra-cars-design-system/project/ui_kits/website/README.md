# Altorra Cars — Website UI Kit

## Overview
Interactive click-through prototype of the Altorra Cars marketing website.
Built from the codebase at `altorracars.github.io/`.

## Screens Covered
1. **Homepage** — Hero, trust bar, vehicle grid, brand logos, categories
2. **Search / Catalog** — Filters sidebar, vehicle results grid
3. **Vehicle Detail** — Gallery, specs, price, CTA, WhatsApp contact
4. **Contact** — Form + info
5. **Auth Modal** — Login / Register

## Components
- `Header.jsx` — sticky nav with dropdowns, auth buttons, favorites badge
- `Footer.jsx` — 3-col dark footer with contact + social
- `HeroSection.jsx` — full-bleed hero with search bar, animated badge, trust bar
- `VehicleCard.jsx` — glassmorphism dark card (new/used/offer states)
- `SearchFilters.jsx` — sidebar filters (brand, type, price range)
- `VehicleDetail.jsx` — full detail view with image gallery

## Usage
Open `index.html` for the full interactive prototype.

## Design Notes
- All screens use the dark theme (#0a0a0a base) with gold (#d4af37) accents
- Font: Poppins from Google Fonts
- Icons: inline SVG, stroke-based
- Language: Spanish (es_CO)
